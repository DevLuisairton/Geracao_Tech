Introdução ao HTML

HTML é a linguagem usada para criar páginas da web. Com as tags semânticas, é possível deixar o conteúdo mais organizado e acessível.

Dicas de Boas Práticas

  • Use tags semânticas sempre que possível.
  • Mantenha a estrutura do código clara.
  • Inclua textos alternativos em imagens.
